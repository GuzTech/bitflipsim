# README #

**bitflipsim** is a gate level simulator that keeps track of the number of toggled bits in a system.

### How do I build it? ###
```
git clone --recursive https://bitbucket.org/guztech/bitflipsim
cd bitflipsim
make init
make
```

### How do I run it? ###
```
./bitflipsim <configuration_file>.yaml
```
